# 🐍 Python Projects

This repository contains a collection of **beginner to intermediate Python projects** created to practice programming concepts and improve problem-solving skills.

Each project focuses on different aspects of Python such as **loops, conditional statements, functions, modules, and API usage**. The goal is to gradually build practical programming experience by developing small but useful projects.

---

## 📂 Projects Included

### 1️⃣ Number Guessing Game

A simple interactive command-line game where the computer generates a random number and the user tries to guess it. The program provides hints (**Too High / Too Low**) after each attempt until the correct number is guessed.

### 2️⃣ Weather App

A Python command-line application that fetches **real-time weather data** for any city using a weather API. It displays temperature, humidity, weather condition, and wind speed.
