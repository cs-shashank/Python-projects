# Number Guessing Game

## Description

The Number Guessing Game is a simple Python-based interactive program where the computer randomly selects a number within a given range (1–100). The player tries to guess the number, and the program provides hints after each attempt indicating whether the guess is too high or too low. The game continues until the correct number is guessed.

This project is designed for beginners to understand basic programming concepts and logic building using Python.

## Features

* Random number generation
* User input handling
* Hint system (Too High / Too Low)
* Attempt counter to track the number of guesses
* Simple and interactive command-line gameplay

## Technologies Used

* Python
* Random module

## How the Game Works

1. The computer generates a random number between 1 and 100.
2. The player enters a guess.
3. The program compares the guess with the secret number.
4. If the guess is too high or too low, the program provides a hint.
5. The player continues guessing until the correct number is found.
6. The program displays the number of attempts taken.

## How to Run the Program

1. Make sure Python is installed on your system.
2. Clone the repository or download the project files.
3. Open a terminal in the project folder.
4. Run the program using the command:

python Numberguessinggame.py
