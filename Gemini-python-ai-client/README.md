# Gemini Python ai client

A simple Python project that demonstrates how to use the **Google Gemini API** to generate AI responses from user prompts.

## 🚀 Features

* Accepts user input from the terminal
* Sends the prompt to the Gemini AI model
* Displays the AI-generated response
* Beginner-friendly implementation

## 🛠 Technologies Used

* Python
* Google Gemini API
* google-genai Python SDK

## 📦 Installation

Clone this repository:

```
git clone https://github.com/yourusername/gemini-python-prompt-generator.git
```

Navigate to the project directory:

```
cd gemini-python-prompt-generator
```

Install required dependencies:

```
pip install -r requirements.txt
```

## ▶️ Usage

Add your Gemini API key in the Python file:

```
client = genai.Client(api_key="YOUR_API_KEY")
```

Run the program:

```
python main.py
```

## 💡 Example

```
Enter your prompt: What is Artificial Intelligence?
```

Output:

```
Artificial Intelligence (AI) refers to the simulation of human intelligence in machines that are programmed to think and learn.
```

## 🎯 Purpose of the Project

This project was created to learn how to integrate **Google's Gemini AI API with Python** and generate responses from user prompts.
